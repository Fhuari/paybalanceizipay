# Plugin de pago con Izipay para cualquier monto

**Izipay** es un plugin de WordPress que permite integrar pagos en línea mediante la pasarela **Izipay Perú**.  
Su objetivo es brindar una solución sencilla, rápida y segura para aceptar pagos en tu sitio web.

---

## 🚀 Características
- Integración con **Izipay Checkout**.
- Compatible con **WordPress Shortcodes**.
- Panel de configuración en el **Administrador de WordPress**.
- Interfaz de usuario construida con **Vue.js** + **Ant Design**.
- Almacenamiento seguro de credenciales (API Key y Secret).
- Soporte para pagos con **tarjeta de crédito y débito**.
